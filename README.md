# keyboard

i think i need a new keyboard, so whynot make a new one. hehehe

## Features
- 60% keyboard - 60 keys
- made using qmk
- uses pi pico
- simple matrix design
- easy to recreate

Also my home is like in a region where its harder to ship stuff so maybe robu might also charge extra
